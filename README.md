# AI Chat Bot Assistance

A simple AI terminal chat assistant powered by OpenAI's GPT models.

## Features
- Real-time interaction with AI.
- Command to clear the chat history.
- Simple terminal interface.

## Setup
1. Clone the repository.
2. Create a `.env` file and add your `OPENAI_API_KEY`.
3. Install dependencies:
   ```bash
   pip install openai python-dotenv
   ```
4. Run the application:
   ```bash
   python AI_CHAT.PY
   ```
